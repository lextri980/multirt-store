# MultirtStore Projects

### Abstract
This is a website to buy and sell items

### Prerequisites
Before you continue, ensure you meet the following requirements:
- Installing a Code editor (ex: VS Code)
- Installing Nodejs
- Install MongoDB

### Installation 
#### 1. Client folder
- Open Terminal
- Run `npm i`
- Create a `.env` file following the `.env.example` file
- Run `npm start`
#### 2. Server folder
- Open Terminal
- Run `npm i`
- Create a `.env` file following the `.env.example` file
- Run `npm start`

### Technologies
#### 1. Client folder
- Framework: `ReactJS`
- State management: `Redux` `Redux Saga`
- Form handler: `React-hook-form` `Yup`
- UI lib: `NextUI` `Material UI`
- Style: `SCSS` `Styled-components`
- Date formater: `moment`
#### 2. Server folder
- Framework: `ExpressJS`
- Database: `MongoDB` `Mongoose`
- Auth: `Jsonwebtoken`
- Image saver lib: `Cloudinary` `Multer`
- Mail handler: `Nodemailer`
